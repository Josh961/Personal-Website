ch@

ch@ is a chat room application built with Ruby on Rails. Our group used Agile, UML Diagrams, SRS documents, and other best practices within the duration of the project. It was a great way to mock how things are done in the industry. We became more efficient with the software development life cycle, along with gaining a familiarity with rails and MVC. Code can be found on GitHub.

Code Bee Bop

I wanted to make a "quick start" guide for getting someone set up with a good looking webpage from scratch. Code Bee Bop quickly gives links to tools, provides code to follow, and is easy to read. Each lesson is short and conveys only what's needed, choosing simplicity over complexity. Code can be found on GitHub.

Programmer Ipsum

A fun little tool to generate "lorem ipsum" text specifically for programmers. Programmer Ipsum was the first project I made to get started in web development. Vanilla HTML, CSS, and JavaScript was used to learn core concepts, along with helping to keep the site fast and simple.
